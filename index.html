ℹ️ How to update this directory
To add, edit, or remove a listing:
  1. Open businesses.csv in Excel or Google Sheets
  2. Make your changes — add rows, edit cells, or delete rows
  3. Save as CSV (Comma delimited) — keep the exact filename businesses.csv
  4. Upload it to your server in the same folder as this HTML file (overwrite the old one)
  5. Refresh the site — changes show instantly

CSV column order:
Category Name Phone Phone2 Website Address Hours Description ReferredBy OGSDad

For the OGSDad column: type Yes to show the OGS Dad badge, leave blank otherwise.
Wrap any field containing commas in double quotes — Excel handles this automatically.
